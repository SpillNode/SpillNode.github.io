Velg Spill Server

Et helt sett med funksjoner og alternativer

Når det gjelder små servere eller store nettverk, har SpillNode dekket deg med våre fleksible, kraftige og rimelige spillhotellplaner.

Hva er inkludert?

  • High clock speed DDR4 memory
  • Intel® Core™ i7 Prosessorer.
  • Rask SSD/HDD lagring
  • Unmetered MySQL® Databases1
  • Backups
  • 98% Oppetid

Intuitiv og tilpasset ledelse

SpillNode gjør det enkelt å administrere spillservere ved å bruke vårt (tilpassede) Pterodactyl panel.

  • Power-handlinger og sanntidskonsoll for å se hvilke spillere som er online.
  • Ressursbruksrapport i sanntid for å se hvordan serverne dine fungerer.
  • Tofaktorautentisering for å holde kontoen din sikker.
  • Full SFTP -tilgang og nettleserfilbehandling for enkelt å laste opp og laste ned serverfiler.
  • Subbrukerkontoer for å la andre styre serverne dine.
